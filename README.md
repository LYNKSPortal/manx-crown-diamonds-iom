# Manx Crown Diamonds

A beautiful, modern website for Manx Crown Diamonds - a bespoke jewellery business based in the Isle of Man, specializing in IGI and GIA Certified diamonds.

## Features

- 🎨 Modern, elegant design with Tailwind CSS
- 💎 Showcase of jewellery collections (Rings, Necklaces, Earrings, Bracelets)
- 🛠️ Services section (Sourcing, Consulting, Customisation, Gift Cards)
- 📱 Fully responsive design
- ⚡ Built with Next.js 14 and TypeScript
- 🎯 SEO optimized

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

### Build for Production

```bash
npm run build
npm start
```

## Customization

The website uses custom colors defined in `tailwind.config.ts`:
- `diamond-blue`: #1e3a8a
- `diamond-gold`: #d4af37
- `manx-red`: #dc2626

You can modify these colors or add new ones in the Tailwind configuration file.
